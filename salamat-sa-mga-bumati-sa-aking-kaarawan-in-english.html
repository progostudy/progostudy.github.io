Pagkatapos ng ISANG salu-salo ,nagkaroon ng palaro. Isang ARAW pabitin  ang ginawa  ng aking tatay.Nagsabit dito ang aking ate ng MAG laruan. Ang MABAIT kong kaibigan ang nakaabot ng bola sa pabitin. Ang buong mag-anak ay NAKIAPAG subalit’t nasiyahan sila dahil naging masaya ang lahat.      

Ang talata o talata ay isang ideya sa anyo ng isang serye ng mga pangungusap na magkakaugnay sa bawat isa. Ang isa pang pangalan para sa mga talata ay mini-discourse. Ang layunin ng mga talata ay markahan ang pagsisimula ng isang bagong paksa at paghiwalayin ang iba't ibang pangunahing ideya.

Ang paggamit ng mga talata ay nagpapadali para sa mga mambabasa na maunawaan ang binasa sa kabuuan. Ang haba ng isang talata ay ilang pangungusap. Ang bilang ng mga pangungusap sa isang talata ay natutukoy sa pamamagitan ng pagbuo at pagkakumpleto ng paglalarawan ng mga ideyang inihahatid.

Maaaring matukoy ng bilang ng mga pangungusap sa isang talata ang kalidad ng pagbasa. Ang mga talata ay binubuo ng pangunahing ideya sa paksang pangungusap. Dagdag pa rito, may mga pangungusap na nagpapaliwanag na nagbibigay linaw sa paksang pangungusap.

Ang mga talata ay gumaganap din ng sistematikong pagpapahayag ng kaisipan ng may-akda upang madaling maunawaan ng mga mambabasa. Ang pamantayan para sa isang set ng mga pangungusap na maaaring maging mga talata ay ang pagkakaroon ng pagkakaisa, pagkakaugnay-ugnay, pagkakumpleto, pagkakaugnay-ugnay, at hindi nagbabagong pananaw.
