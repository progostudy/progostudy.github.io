21. Find a colloquial word in i gotta learn this for my examination


Gotta is an example of a colloquial word.

If you are not familiar with the definition of a colloquial word, it is the type of word which is not used in a formal setting. You might have heard or seen the examples, though: gotta, wanna, imma, etc. These words happen to be the abbreviation, too, such as have to as in gotta, wanna is the short version of want to, and imma is I am going to. T

herefore, in the sentence: I gotta learn this for my examination, the colloquial word is gotta. If it's rewritten so we can use the same sentence in a formal setting: I have to learn this for my examination.
