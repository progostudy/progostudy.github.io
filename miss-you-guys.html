17. A.solve for the other missing length given only two measurement.can you help me guys.​


I will only solve one problem since this will get real messy. Medyo mahaba solution ko kaya isang item lang sasagutin ko para malaman mo kung ano 'yung mga gagawin.

1. a = 4, b = 6

We will first find c by Pythagorean theorem:

[tex]c=\sqrt{(a^2+b^2)} \\c=\sqrt{(4^2+6^2)} \\c=\sqrt{(16+36)} \\c=\sqrt{52} \\c= 7.21[/tex]

Since we know c, we can solve for h, n, and m.

To find h, we will have a rather complicated way of solving. We will need 2 formulas: Heron's formula and the formula for the height of the triangle. First, we will use Heron's formula since we know the lengths of the outer sides (a, b, and c)

[tex]A=\sqrt{s(s-a)(s-b)(s-c)} \\where\\s=\frac{a+b+c}{2}[/tex]

We will first find s.

[tex]s=\frac{a+b+c}{2} \\s=\frac{4+6+7.21}{2} \\s=8.605[/tex]

We can now use s to solve for the area of the triangle because this will be needed when we solve for h.

[tex]A=\sqrt{s(s-a)(s-b)(s-c)} \\A=\sqrt{8.605(8.605-4)(8.605-6)(8.605-7.21)} \\A=\sqrt{8.605(4.605)(2.605)(1.395)} \\ A=\sqrt{144} \\A=12[/tex]

We can now use the formula for the height of the triangle.

[tex]h=2(\frac{A}{b})\\h=2(\frac{12}{7.21})\\h=2(1.66)\\h=3.33[/tex]

We now know h. To find n, we will use the Pythagorean theorem on b (hypothenus) and h.

[tex]n=\sqrt{b^2-h^2} \\n=\sqrt{6^2-3.33^2} \\n=\sqrt{24.91} \\n=5[/tex]

We will subtract c to n to get m. (or, if you want a harder but more accurate result, use Pythagorean theorem.)

[tex]m = c-n\\m=7.21-5\\m=2.21[/tex]

The values in number 1 are:

c - 7.21

h - 3.33

n - 5

m - 2.21
