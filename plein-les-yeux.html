9. Question given: Quadrilateral LIKE is a kite with line LI≈LK and line LE ≈KE. A. LE is twice LI. If it's perimeter is 21 cm, how long is line LE?


I have encountered this problem before. This was my answer.

Given:
LI is congruent to IK ⇒ LI = IK
LE is congruent to KE ⇒ LE = KE
LE is twice LI ⇒ LE = 2LI

Perimeter = LI + IK + LE + KE
21 cm = 2LI + 2(2LI)
21 cm = 6LI
21 cm/6 = LI
3.5 cm = LI

LI = IK : LI = 3.5 cm ; IK = 3.5 cm
LE = KE: LE = 2LI ; LE = 2(3.5 cm) ; LE = 7 cm; KE = 7cm

Perimeter = LI + IK + LE + KE
21 cm = 3.5 cm + 3.5 cm + 7 cm + 7 cm
21 cm = 21 cm

19. if LE = 3× +4, OV =4, find x, LE and OV​


The given equation is:

LE = 3x + 4

And OV = 4.

To solve for x, we need more information or an additional equation. However, if we assume that OV is a point on the line LE, then we can find the value of x by substituting the value of OV into the equation for LE:

OV = LE

4 = 3x + 4

Subtracting 4 from both sides gives:

0 = 3x

Dividing both sides by 3 gives:

0 = x

So, x = 0.

Substituting x = 0 into the equation for LE gives:

LE = 3(0) + 4 = 4

Therefore, LE = 4 and x = 0 and OV = 4.

25. Question given: Quadrilateral LIKE is a kite with line LI≈LK and line LE ≈KE. A. LE is twice LI. If it's perimeter is 21 cm, how long is line LE?


Let LI be x and LE is 2x.
Perimeter = 21 cm

Perimeter of kite LIKE = 2 (LI) + 2 (LE)
     21 = 2 (x) + 2 (2x)
     21 = 2x + 4x
     21 = 6x
    
     6x/6 =21/6
     x = 7/2

Plug-in 7/2 to x:
    LI = x          ⇒   LI = 7/2 cm
    LE=2x        ⇒    LE = 2(7/2) = 7 cm

ANSWER: LE is 7 cm.

Check: 
     7/2 + 7/2 + 7 + 7 = 21
     14/2 + 14 = 21
      7 + 14 = 21
      21 = 21   (True)

In relational art, the audience is envisaged as a community. Rather than the artwork being an encounter between a viewer and an object, relational art produces encounters between people. Through these encounters, meaning is elaborated collectively, rather than in the space of individual consumption.
