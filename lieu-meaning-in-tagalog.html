8. Which set of three words do not ALL rhyme with each other: 1. flu / lieu / who 2. reigns /stains /reins 3. draft / draught / aught 4. parade / weighed / wade


Answer:

3. Draft / Draught / Aught

Explanation:

1. Flu - floo

Lieu - loo

Who - hoo

2. Reigns - rāns

Stains - stāns

Reins - rāns

3. Draft - draft

Draught - draft

Aught - ôt

4. Parade - pəˈrād

Weighed - wād

Wade - wād

All the words in the first number have the same sound ( oo sound).

All the words in the second number have the same sound too ( ā or ey sound).

All words in the third number sound the same except for the Aught ( a sound, Aught sounds ô).

All the words in the fourth number have the same sound ( ā or ey sound).

Therefore the answer to the question "Which set of three words do not ALL rhyme with each other?" is number 3.

Rhyme is the same sound between words or the endings of words.
